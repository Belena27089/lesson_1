# Дата создания: октябрь 2016
##################
# Команда разработчиков: Simple Solutions
##################


Сайт предназначен для любителей экстремального отдыха и острых ощущений. На сайте представлена информация о местах для занятий экстримом, с возможностью поиска по городу/местоположению, категории, дате добавления и рейтингу. Для зарегистрированных пользователей есть возможность добавлять на карте собственные места для занятий экстримом (с их описанием и фото), оставлять комментарии, оценивать и проставлять рейтинг мест.
